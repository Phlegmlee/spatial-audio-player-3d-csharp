# Spatial Audio Extended
![Static Badge](https://img.shields.io/badge/REWRITE_STATUS-IN_PROGRESS_0%25-red)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE.md)

Spatial Audio Extended provides an advanced, drop-in replacement, SpatialAudioPlayer3D, for Godot's AudioStreamPlayer3D. Originally created by Danikakes, it provides physically-inspired, real-time spatial audio effects for immersive 3D games. 

_Note: This is a C# rewrite of [spatial_audio_player_3d](https://github.com/Danikakes/spatial_audio_player_3d) by Danikakes._
